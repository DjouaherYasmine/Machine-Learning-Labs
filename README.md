# Machine Learning Labs 

## Overview
This repository contains a collection of laboratory works completed as part of coursework in Machine Learning and Data Science. Each notebook focuses on a fundamental algorithm or concept, combining theoretical foundations with practical implementation.

## Repository Structure

- **Lab01 – Linear Regression (LR)**  
  Introduction to supervised learning using linear regression models.

- **Lab02 – Naive Bayes (NB)**  
  Probabilistic classification based on Bayes’ theorem.

- **Lab03 – Decision Trees (DT)**  
  Tree-based models for classification and interpretability.

- **Lab04 – Support Vector Machines (SVM)**  
  Margin-based classification and kernel methods.

- **Lab05 – Neural Networks (NN)**  
  Fundamentals of neural networks and basic deep learning concepts.

- **Lab06 – Regularization and Feature Selection**  
  Techniques to improve generalization and reduce overfitting.
